# Amazon Sales Dashboard

## Project Overview

This project analyzes Amazon product sales data to identify trends in product performance, customer engagement, pricing strategies, and category-wise insights using Power BI.

The goal of this project is to perform data cleaning, transformation, and visualization to create an interactive dashboard useful for business decision-making.

---

## Tools Used

* Power BI
* Power Query
* Microsoft Excel
* Data Cleaning
* Data Visualization

---

## Dataset Information

The dataset contains product-level information including:

* Product Name
* Category
* Discounted Price
* Actual Price
* Discount Percentage
* Rating
* Rating Count
* Product Reviews

---

## Data Cleaning Performed

* Removed unnecessary columns
* Cleaned rating column errors
* Split category column using delimiter "|"
* Extracted main category
* Cleaned product names using delimiters
* Created custom percentage column for discount analysis
* Applied Top N filtering for popular product analysis

---

## Dashboard Visualizations

### 1. Category Performance

Shows top-performing product categories.

### 2. Top Rated Products

Displays highest-rated products based on customer ratings.

### 3. Price by Category

Compares average discounted price across categories.

### 4. Discount vs Rating

Analyzes whether discounts impact product ratings.

### 5. Most Popular Products

Shows Top 10 products based on rating count.

---

## Key Insights

* Electronics and accessories dominate product listings
* Highly rated products often have strong review counts
* Discounts influence customer engagement
* Certain categories show higher pricing trends
